# 專題目的
感於近年來受疫情影響，國人消費形態及支付方式改變，網路交易及外送盛行，希望藉由國人消費様態分析作為發卡及收單機構推廣特約商店及行銷推廣之參考，並預估下月份之信用卡消費金額。
